# jssip-polymer
jssip + polymer

# Intall polymer

npm install -g polymer-cli

# Install bower

npm install -g bower

# Intall bower components

bower install

# Run 

polymer serve


 Start SIP server
 
 To be able to make a call you have to have your own sip server(can be local). You can use this server https://www.kamailio.org/wiki/install/stable/debian
